Detailed Description

4x4 Matrix.

A 4x4 Matrix with column-major order.

Note
Linear memory storage layout: [0] [4] [8 ] [12] [1] [5] [9 ] [13] [2] [6] [10] [14] [3] [7] [11] [15]
